# 🏥 Hospital Management System Database

Kani waa mashruuc Relational Database ah oo aan u dhisay si loo maamulo xogta cisbitaal dhab ah.

## 🛠️ Tiknoolajiyada la isticmaalay:
- **MS SQL Server (SSMS)**
- **SQL (DDL & DML)**

## 📊 Naqshadda Database-ka
Mashruucu wuxuu ka kooban yahay saddexda table ee ugu muhiimsan:
1. `Patients` - Lagu kaydiyo xogta bukaannada.
2. `Doctors` - Lagu kaydiyo dhakhaatiirta.
3. `Appointments` - Xiriirka u dhexeeya bukaanka iyo dhakhtarka.
